# Predictive Maintenance and Anomaly Detection in 6G-Integrated Smart Manufacturing Systems

## Overview
This project predicts machine maintenance risks using anomaly detection and sensor analytics in a 6G-enabled smart manufacturing environment.

## Features
- Predictive maintenance dashboard
- Machine-wise monitoring
- Risk classification (Low / Medium / High)
- Anomaly detection using Isolation Forest
- Sensor trend visualization

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset Features
- Temperature
- Vibration
- Power Consumption
- Network Latency
- Packet Loss
- Error Rate
- Production Speed
